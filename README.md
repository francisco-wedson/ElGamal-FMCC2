## Descrição do Projeto
### O projeto consiste em uma implementação do esquema criptográfico ElGamal, concebido em 1985 por Taher ElGamal. 
### O modelo é elaborado com base na dificuldade em encontrar logaritmos com módulos de números primos grandes e no esquema Diffie-Hellman para troca de chaves. 
### Além disso, este projeto apresenta uma comparação entre o algoritmo apresentado e outros sistemas criptográficos semelhantes.
